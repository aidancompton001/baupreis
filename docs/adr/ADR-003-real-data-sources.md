# ADR-003: 7 реальных API источников вместо синтетических данных

**Дата:** 2026-02-26
**Статус:** accepted
**Автор:** #5 Data Pipeline Engineer

## Контекст

MVP использовал синтетические данные (random walk). Для production нужны реальные цены на 16 стройматериалов в Германии. Платные API (metals.dev, steelprices.eu) дорогие.

## Варианты

1. **Платные API** (metals.dev $50/мес, steelprices.eu) — точные данные, дорого
2. **Бесплатные государственные API** (Eurostat, Destatis, SMARD) — бесплатно, индексы вместо абсолютных цен
3. **Гибрид** — бесплатные для основных категорий + один платный (metals.dev) для металлов

## Решение

Выбран вариант 3 (гибрид):
- Eurostat COMEXT — стройматериалы (бетон, кирпич, изоляция, стекло, древесина)
- SMARD — электричество (реальная биржевая цена)
- metals.dev — металлы (сталь, медь, алюминий, цинк)
- ECB — обменные курсы для конвертации
- Всего 7 API, покрытие 16/16 материалов

## Последствия

- (+) Реальные данные, 0 синтетики
- (+) Большинство источников бесплатно (Eurostat, SMARD, ECB)
- (-) Индексы вместо абсолютных цен для некоторых материалов
- (-) Зависимость от доступности государственных API (риск #1, #6)
- (-) metals.dev — единственный платный, нужен fallback
