## 3. РЕКОМЕНДАЦИИ ДЛЯ ПОДЪЁМА С 6/10 ДО 8-9/10

### Почему 6, а не 8

| Фактор | Сейчас (6/10) | Нужно для 8-9/10 |

| Платящие клиенты | 0 | 3-5 |

| Данные | Частично synthetic | Real sources работают |
| AI прогнозы | $4.78 на Anthropic | Бюджет на 6 мес или fallback |
| Обратная связь | Нет | 5+ интервью с ЦА |

### УРОВЕНЬ 1: Без кода, без денег (СЕЙЧАС, +1.5 балла)

**1. Получить 5 LOI (Letter of Intent) от Bauunternehmen.**
Найти 20 Einkaufer/Bauleiter в Мюнхене через IHK-Firmendatenbank или LinkedIn. Написать: "Ich entwickle ein Tool zur automatischen Baustoffpreisueberwachung. Darf ich Ihnen 15 Minuten zeigen und Ihr Feedback bekommen?" Из 20 согласятся 3-5. Показать live-демо. Спросить: "Wuerden Sie 49 EUR/Monat dafuer bezahlen?" 5 LOI = доказательство WTP.

*"Get out of the building." -- Steve Blank*

**2. Один кейс-расчёт: сколько экономит BauPreis.**
Взять реальные данные из базы (3100+ цен). Конкретный сценарий: "Bauunternehmen X закупает 200т Betonstahl в месяц. BauPreis-алерт 15.02 показал снижение цены на 4%. Закупка на 2 дня раньше = экономия 3.200 EUR." Один конкретный кейс > все маркетинговые обещания.

**3. Написать Oliver Nerz (IHK) до 11 марта.**
Не просто Tragfaehigkeitsbescheinigung — попросить контакт с IHK-Firmendatenbank или Bauinnung Muenchen. IHK = канал доступа к клиентам. Один тёплый intro от IHK > 100 холодных писем.

### УРОВЕНЬ 2: Платформа — точечные изменения (2-4 недели, +1 балл)

**4. Бесплатный публичный виджет: "Stahlpreis heute".**
Одна страница, без регистрации, SEO-оптимизирована. Текущая цена Betonstahl + график 30 дней. Внизу CTA: "Alle 16 Baustoffe + KI-Prognose -> 14 Tage kostenlos testen." Lead magnet 24/7.

**5. Отчёт-PDF: "Baustoffpreise Q1 2026 — Marktanalyse".**
Автоматически из данных в БД. 3-4 страницы. Скачивание за email. Контент-маркетинг без бюджета.

**6. Real data pipeline хотя бы для 3 ключевых материалов.**
Betonstahl, Kupfer, Zement — три самых волатильных. Реальные цены из metals.dev + публичный источник.

### УРОВЕНЬ 3: Бизнес-модель (3-6 месяцев, +1 балл)

**7. Первый платящий клиент.**
Через личную продажу. Найти одного Einkaufer из LOI-списка. Настроить dashboard, подключить 5 материалов, настроить алерты. Взять 49 EUR хоть переводом. Один платящий клиент = бизнес существует.

*"Revenue is the ultimate validation." -- Jason Lemkin, SaaStr*

**8. Freemium tier: 1 материал бесплатно, навсегда.**
Снимает барьер входа полностью. Пользователь видит ценность -> upgrade. PLG > outbound для solo-founder.
