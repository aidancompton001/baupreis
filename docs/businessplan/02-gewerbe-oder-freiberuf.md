# Gewerbe или Freiberuf (Freie Berufe)?

## В чём разница?

| | **Gewerbe** | **Freiberuf** |
|---|---|---|
| **Регистрация** | KVR (Gewerbeamt) + Finanzamt | Только Finanzamt |
| **Стоимость** | ~50-60€ | Бесплатно |
| **IHK членство** | Обязательное (платное) | Нет |
| **Gewerbesteuer** | Да (с прибыли > €24.500) | Нет |
| **Бухгалтерия** | EÜR (до €600K оборота) | EÜR |
| **Торговый реестр** | Не обязательно (Kleingewerbe) | Нет |

## Как IT-разработка классифицируется?

IT-деятельность может быть **и тем, и другим** — зависит от характера работы:

### Freiberuf (свободная профессия):
- **Разработка ПО** на заказ (индивидуальные решения)
- **IT-консалтинг** (Beratung)
- **Системное проектирование** (Systemarchitektur)
- Требуется: квалификация (диплом или доказуемый опыт)

### Gewerbe (предпринимательство):
- **Продажа готового ПО** (SaaS = продажа подписок!)
- Посредничество, перепродажа
- Стандартизированные услуги

### Для BauPreis:
**BauPreis = SaaS = продажа подписок = скорее всего Gewerbe**

Ты продаёшь доступ к платформе за ежемесячную плату — это торговая деятельность, а не индивидуальная интеллектуальная работа.

> **Однако:** Если ты позиционируешь себя как IT-консультант, который также предоставляет доступ к своему инструменту — можно попробовать аргументировать Freiberuf. Финальное решение — за Finanzamt.

---

## Как определить точно?

1. **Спросить IHK München** на бесплатной консультации — они скажут своё мнение
2. **Обратиться в Institut für Freie Berufe** (IfB Nürnberg) — дадут заключение
3. **Finanzamt** решает окончательно — при подаче Fragebogen zur steuerlichen Erfassung

---

## Рекомендация для тебя

**Регистрируй Gewerbe (Kleingewerbe).** Причины:
- SaaS-подписки однозначно коммерческая деятельность
- Kleingewerbe — простая регистрация, минимум бюрократии
- При обороте до €24.500/год Gewerbesteuer = 0
- IHK-взнос для Kleingewerbe минимальный (~0€ первый год)
- Gewerbeschein — документ, который примут Paddle, Stripe, банки
