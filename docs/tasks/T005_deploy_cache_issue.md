## Roadmap

### Вариант A: Принудительный refresh (быстрый, 5 сек)
1. CEO делает **Ctrl+Shift+R** (Windows) или **Cmd+Shift+R** (Mac) в браузере
2. Это пропускает кэш и загружает новые файлы

### Вариант B: Очистка кэша (если Ctrl+Shift+R не помогло)
1. В браузере: Settings → Privacy → Clear browsing data
2. Выбрать: Cached images and files
3. Нажать Clear data
4. Перейти на baupreis.ais152.com

### Вариант C: Добавить cache-busting headers на сервере (превентивно)
1. Проверить Caddy configuration на предмет cache headers
2. Добавить `Cache-Control: no-cache` для HTML (index)
3. Статические ассеты (.js, .css) уже имеют хэш в имени — кэширование ОК
4. Перезапустить Caddy
